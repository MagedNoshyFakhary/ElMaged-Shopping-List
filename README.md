# ElMaged-Shopping-List
This app is intended to save the things to be purchased

## Getting Started
This app depends on the firebase database with a copy saved in a Sqlite local database


## Built With
* Firebase Auth
* Firebase RecyclerView
* Firebase RealtimeDB
* Sqlite DB
* Material Design 
* DrakView
